# 前端模块

## 组件

| 组件名称              | 描述                                                                             | 主要参数                                 |
| --------------------- | -------------------------------------------------------------------------------- | ---------------------------------------- |
| `DownloadFileButton`  | 一个用于触发文件下载并显示加载动画的按钮组件。                                     | `url`, `params`, `children`              |
| `LinkButton`          | 一个点击后可以导航到指定路径的按钮组件。                                         | `path`, `label`, `children`, `size`      |
| `Ellipsis`            | 一个截断文本并用省略号表示的组件，点击后可在模态框中显示完整文本。             | `length`, `children`, `pre`              |
| `NamedIcon`           | 一个根据 `name` 属性动态渲染 Ant Design 图标的组件。                           | `name`                                   |
| `PageLoading`         | 一个显示页面加载中动画和可自定义消息的组件。                                     | `message`                                |
| `FieldBoolean`        | 布尔值输入组件，提供 `select`, `radio`, `checkbox`, `switch` 等显示类型。      | `value`, `onChange`, `type`              |
| `FieldDate`           | 日期/时间输入组件，根据 `type` 属性自动渲染不同的 Ant Design 选择器。          | `type`, `value`, `onChange`              |
| `FieldDictSelect`     | 字典选择组件，从后端字典数据加载选项的下拉框。                                   | `value`, `onChange`, `typeCode`          |
| `FieldRemoteSelect`   | 远程搜索选择组件，通过异步请求从远程 URL 加载选项并支持搜索。                  | `url`, `value`, `onChange`, `placeholder`|
| `FieldRemoteSelectMultiple` | 远程多选组件，从远程 URL 异步加载选项并支持多选。                     | `url`, `value`, `onChange`, `placeholder`|
| `FieldTableSelect`    | 表格选择组件，通过表格形式展示选项，支持单选或多选。                           | `url`, `value`, `onChange`, `columns`    |
| `FieldUploadFile`     | 文件上传组件，用于文件上传操作。                                                 | `url`, `value`, `onChange`, `maxCount`   |
| `FieldEditor`         | 富文本编辑器组件。                                                               | `value`, `onChange`                      |
| `FieldPercent`        | 百分比输入组件。                                                                 | `value`, `onChange`                      |
| `FieldSysOrgTree`     | 系统机构树选择组件。                                                             | `value`, `onChange`                      |
| `FieldSysOrgTreeSelect` | 系统机构树选择组件（选择器模式）。                                                 | `value`, `onChange`                      |
| `FieldRemoteTree`     | 远程树形结构选择组件。                                                           | `url`, `value`, `onChange`               |
| `FieldRemoteTreeCascader` | 远程树形级联选择组件。                                                           | `url`, `value`, `onChange`               |
| `FieldRemoteTreeSelect` | 远程树形选择器组件。                                                             | `url`, `value`, `onChange`               |
| `FieldRemoteTreeSelectMultiple` | 远程树形多选器组件。                                                             | `url`, `value`, `onChange`               |
| `ViewBoolean`         | 布尔值展示组件。                                                                 | `value`                                  |
| `ViewPassword`        | 密码展示组件。                                                                   | `value`                                  |
